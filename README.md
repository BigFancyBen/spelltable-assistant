### Install Dependencies

```

# using yarn or npm
$ yarn (or `npm install`)

```

### Use it

```
# development mode
$ yarn dev (or `npm run dev` or `pnpm run dev`)

```


Download the Bulk Data file from scryfall and place it in /renderer/public/scryfall/cards.json
